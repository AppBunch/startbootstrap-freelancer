App for Google Reader

AppGReader is a full-screen browser that displays Google Reader at start. The function is simple, the same as Safari.



FEATURES:

- Browsing Google Reader on a large screen.

- Remembering a google account besides Safari.

- A built-in browser that displays links.

- Posting to Facebook.

- Posting to Twitter.

- Interface orientation settings.

- Reloading the last URL at start up.

- Initializing the URL by shaking

- Custom URL scheme:

AppGReader:,[OpenURL at Start],[OpenURL at End]




ATTENTION:

- This is not a proprietary application for Google Reader.
